탕바오 汤包

가오더지투 (高德地图) 앱 활용법

중국 현지 지도·내비게이션·택시 호출까지 하나로 해결하는 필수 앱. 아래 순서대로 설정하면 현지에서 바로 활용 가능.
1
수집함(收藏夹) 만들기
하단 我的 탭 → 收藏夹 → 우측 상단 创建收藏夹
이름을 "南京旅行"으로 설정. 비공개(私密) 체크 권장.
2
장소 검색 → 수집
상단 검색창에 중국어 이름 입력 (아래 검색 목록 참조) → 검색 결과 터치 → 상세 페이지의 ⭐ 收藏 버튼 → "南京旅行" 수집함에 저장. 24곳 등록에 약 10~15분.
3
현지에서 내비게이션
我的 → 收藏夹 → "南京旅行" → 원하는 장소 터치 → 到这去(여기로 가기) → 도보/지하철/택시 선택.
4
택시 호출 (打车)
수집한 장소 상세 페이지에서 打车 버튼으로 바로 DiDi 택시 호출 가능. 별도 앱 설치 불필요. 알리페이 결제 연동.
5
지도에서 길게 눌러 수집
검색 없이 지도 위 원하는 위치를 길게 누르면(长按) 팝업 → 收藏 선택. 현지에서 발견한 장소를 즉시 저장할 때 유용.
